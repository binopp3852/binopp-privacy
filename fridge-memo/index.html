메모하는 냉장고 개인정보처리방침
최종 수정일: 2026년 4월 9일

binopp(이하 "개발자")은 「메모하는 냉장고」(이하 "앱")을 운영함에 있어 이용자의 개인정보를 중요하게 생각하며, 아래와 같이 개인정보처리방침을 수립합니다.

1. 수집하는 개인정보 항목

가. Google 로그인 시 수집 항목

항목수집 목적보관 기간
Google 계정 ID (sub)회원 식별, 데이터 동기화회원 탈퇴 시 즉시 삭제
이름프로필 표시회원 탈퇴 시 즉시 삭제
이메일 주소계정 관리, 고객 지원회원 탈퇴 시 즉시 삭제

나. 앱 사용 중 생성 데이터

항목내용저장 위치
메모 내용사용자가 직접 입력한 할 일 텍스트기기 로컬 + 서버(로그인 시)
메모 위치·색상·각도포스트잇 배치 정보기기 로컬 + 서버(로그인 시)
냉장고 설정선택한 스킨, 냉장고 이름기기 로컬

다. 자동 수집 정보 (비로그인 포함)

  • 광고 식별자 (GAID): Google AdMob 광고 게재 목적으로 Google이 수집
  • 앱 충돌 정보: 오류 개선 목적 (향후 Firebase Crashlytics 적용 시)

2. 개인정보의 이용 목적

  • 메모 데이터의 기기 간 동기화 및 백업
  • 광고 게재 (Google AdMob 배너·전면·보상형 광고)
  • 앱 기능 개선 및 오류 수정
  • 이용자 문의 대응

3. 제3자 서비스

Google AdMob (광고)

본 앱은 광고 제공을 위해 Google AdMob을 사용합니다. AdMob은 광고 개인화를 위해 다음 정보를 수집할 수 있습니다:

  • 광고 식별자 (Android: GAID)
  • 기기 정보 (모델, OS 버전)
  • 대략적인 위치 정보
  • 앱 사용 데이터

Google 개인정보처리방침 | AdMob 데이터 사용 안내

이용자는 기기 설정에서 광고 맞춤설정을 거부할 수 있습니다.
(Android: 설정 → Google → 광고 → 광고 맞춤설정 선택 해제)

Amazon Web Services (서버)

메모 동기화를 위해 AWS Lambda + DynamoDB (서울 리전)를 사용합니다. 서버에 저장된 데이터는 회원 탈퇴 시 삭제됩니다.

4. 어린이 개인정보 보호

본 앱은 만 14세 미만 아동을 대상으로 하지 않습니다. 만 14세 미만 아동이 개인정보를 제공한 경우, 보호자가 요청하면 즉시 삭제합니다.

5. 데이터 보유 및 삭제

  • 로그인 사용자: 회원 탈퇴 요청 후 30일 이내 서버 데이터 완전 삭제
  • 비로그인 사용자: 앱 삭제 시 기기 내 모든 데이터 즉시 삭제
  • 광고 관련 데이터: Google의 개인정보처리방침에 따라 처리

6. 이용자의 권리

이용자는 언제든지 개인정보 열람, 수정, 삭제 및 처리 정지를 요청할 수 있습니다. 아래 연락처로 요청하시면 10일 이내 처리합니다.

7. 개인정보처리방침 변경

본 방침은 관련 법령 또는 서비스 변경에 따라 개정될 수 있으며, 변경 시 앱 내 또는 스토어 페이지를 통해 사전 공지합니다.
