심리테스트 - Psych Quiz 개인정보처리방침

최종 수정일: 2026년 4월 8일

binopp(이하 "개발자")은 「심리테스트 - Psych Quiz」(이하 "앱")을 운영함에 있어 이용자의 개인정보를 중요하게 생각하며, 아래와 같이 개인정보처리방침을 수립합니다.

1. 수집하는 개인정보 항목

본 앱은 회원가입, 로그인, 서버 전송 없이 동작하며, 이용자의 개인 식별 정보를 수집·저장하지 않습니다.

앱은 다음 데이터를 기기 내부에만 저장합니다:

  • 마지막으로 진행한 테스트 정보 (앱 내 표시 목적)

이 데이터는 기기 밖으로 전송되지 않으며, 개발자도 접근할 수 없습니다.

2. 제3자 서비스 — Google AdMob

본 앱은 광고 제공을 위해 Google AdMob을 사용합니다. AdMob은 광고 개인화를 위해 다음 정보를 수집할 수 있습니다:

  • 광고 식별자 (Android: GAID)
  • 기기 정보 (모델, OS 버전)
  • 대략적인 위치 정보
  • 앱 사용 데이터

Google 개인정보처리방침 | AdMob 데이터 사용 안내

이용자는 기기 설정에서 광고 맞춤설정을 거부할 수 있습니다.
(Android: 설정 → Google → 광고 → 광고 맞춤설정 선택 해제)

3. 결과 공유 기능

테스트 결과 공유 시 기기의 공유 기능(Share Sheet)을 사용합니다. 공유할 대상 앱은 이용자가 직접 선택하며, 개발자는 공유된 내용에 접근하지 않습니다.

4. 어린이 개인정보 보호

본 앱은 만 14세 미만 아동을 대상으로 하지 않습니다. 만 14세 미만 아동이 앱을 이용하는 경우, 보호자의 지도하에 사용하시기 바랍니다.

5. 데이터 보유 및 삭제

앱에서 저장하는 모든 데이터는 기기에서 앱을 삭제하면 즉시 삭제됩니다. 개발자 서버에 별도 저장되는 데이터는 없습니다.

6. 개인정보처리방침 변경

본 방침은 관련 법령 또는 서비스 변경에 따라 개정될 수 있으며, 변경 시 앱 내 또는 스토어 페이지를 통해 사전 공지합니다.
